Your Mission

The page looks acceptable on desktop but fails on smaller screens. Make the typography responsive.

  • Replace fixed oversized heading text with clamp().
  • Add a media query for smaller screens.
  • Set a readable line length using ch.
  • Use unitless line-height.

The Interface Must Work Everywhere

Users may view a website on a phone, tablet, laptop, classroom projector, or large desktop monitor. Responsive typography helps text remain readable across different screen sizes without forcing users to zoom or scroll awkwardly.

Good responsive type uses flexible units, sensible line lengths, and scalable headings. The goal is not just to make text smaller, but to preserve hierarchy, rhythm, and readability.
